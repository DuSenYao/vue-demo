# vue-demo

> 一个用于学习 Vue 的项目

## Project setup

```bash
# 克隆项目
git clone https://github.com/DuSenYao/vue-demo.git

# 启动服务
npm run serve
```

## 发布

```bash
# 构建测试环境
build:stage

# 构建生产环境
npm run build
```

### 其他

```bash
# 代码格式检查
npm run lint

# 代码格式检查并自动修复
npm run lint -- --fix

# 样式格式检查
npm run lint:css
```
